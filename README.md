# IGym - Gym Management System Website
[![Ask DeepWiki](https://devin.ai/assets/askdeepwiki.png)](https://deepwiki.com/yeagx/IGYM---Group)

This repository contains the source code for the IGym landing page, a modern and responsive website designed to market an all-in-one gym management system. The site showcases the features, pricing, and benefits of the IGym software platform for gym owners, trainers, and trainees.

## Key Features

*   **Responsive Design:** The layout is fully responsive and optimized for various devices, from mobile phones to desktops.
*   **Dynamic Hero Section:** An engaging introduction with clear calls-to-action and visuals of the application interface across different devices.
*   **Feature Showcase:** Dedicated sections detailing key software functionalities such as `Insights and Reporting`, `Payment Processing`, and system customization.
*   **Dual App Display:** A unique visual representation comparing the features available for both trainers and trainees in their respective mobile apps.
*   **Comprehensive System Overview:** Highlights the efficiency of the platform, covering membership management, attendance tracking, billing, and staff management.
*   **Interactive Pricing Slider:** A responsive slider powered by Swiper.js presenting various subscription tiers (Basic, Standard, Premium, Ultimate).
*   **Customer Testimonials:** A section to display positive reviews and build social proof.
*   **Contact & Demo Booking:** An integrated contact form and prominent calls-to-action for scheduling a product demonstration.

## Technologies Used

*   **Frontend:** HTML5, CSS3, JavaScript
*   **Frameworks & Libraries:**
    *   Bootstrap 4: For the responsive layout and grid system.
    *   Swiper.js: For the interactive pricing plan slider.
    *   Font Awesome: For icons used throughout the site.
    *   Google Fonts: For the 'Poppins' typography.

## Getting Started

To view the website locally, follow these simple steps:

1.  **Clone the repository:**
    ```sh
    git clone https://github.com/yeagx/igym---group.git
    ```

2.  **Navigate to the project directory:**
    ```sh
    cd iIGym - Gym Management System Websitegym---group
    ```

3.  **Open the website:**
    Open the `index.html` file in your preferred web browser.
